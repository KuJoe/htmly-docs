HTMLy is build to run on very modest and widely offered Webspace-Features: Shared hosting or even free hosting should work as long as the host supports at least PHP 7.2. You do not need to use a VPS (Virtual Private Server) to run it.

* Web server (Apache, Nginx, LiteSpeed, Lightly, IIS, etc.)
* PHP 7.2 or higher
* php-json: Used for reading/writing app data files and API-style responses, including menu data, search index, comments, and admin AJAX payloads.
* php-mbstring: Used for multibyte-safe text handling in content processing and summaries (substring, length, encoding conversion), which is important for non-ASCII languages.
* php-libxml + php-dom + php-simplexml + php-xml: Used for parsing and transforming HTML fragments and for RSS/XML feed import.
* php-hash: Used for secure hashing/HMAC operations, especially in comment subscription token generation and verification flows.
* php-session: Used for login state, CSRF lifecycle, and admin authentication flow.
* php-pcre: Used heavily for URL/content cleanup, validation, and parsing with regex operations across routing/content helpers.
* php-filter: Used for value validation/sanitization patterns such as email validation and boolean checks in some dependency internals.
* php-ctype: Required by PHPMailer internals and used in mail stack compatibility paths.
* php-openssl: Used by updater/HTTPS trust chain and mail security features (secure SMTP and crypto helpers).
* php-zip: Used for backup creation and update package extraction via ZipArchive.
* php-gd: Used for image thumbnail generation and MFA QR image rendering path.
* php-iconv: Used by QR code dependency for encoding conversion when generating QR data.
* php-intl: Used for localized date formatting and related locale-aware output. If missing, app falls back to basic date formatting.
* php-apcu: Used as optional in-memory cache acceleration; app still works without it.
* php-mcrypt: Legacy optional cookie encryption branch; app degrades gracefully when unavailable.

## Important

Since version 2.7.7, a few themes need at least PHP 5.5 to work properly ([info](https://github.com/danpros/htmly/issues/437)).

If you need to run it on PHP 5.3 or PHP 5.4, please download the old themes from previous version (2.7.6): [https://github.com/danpros/htmly/releases/tag/v2.7.6](https://github.com/danpros/htmly/releases/tag/v2.7.6)
